# Signup – Login – Home Page Flow (HTML)

This project demonstrates a simple multi-page website using HTML. It includes a signup page, a login page, and a home page with basic navigation between pages.

## Project Flow

1. **Signup Page**

   * Input fields: First Name, Last Name, Password
   * Submit button redirects to the Login page.

2. **Login Page**

   * Input fields: First Name and Last Name
   * Submit button redirects to the Home page.

3. **Home Page**

   * Displays a greeting message: **"Happy Holi"**.

## Technologies Used

* HTML5
* Basic form elements
* Page navigation using links

## Learning Purpose

This project was created as part of my HTML learning practice to understand form creation and navigation between multiple web pages.
